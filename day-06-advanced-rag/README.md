## 🤖 1. Vector Search Engine (`vector_search.py`)
Implemented an advanced RAG retrieval step using **ChromaDB**. The script initializes a local vector database, ingests document chunks, automatically generates embeddings, and performs a semantic similarity search based on dynamic user input.

## ⚡ 2. DSA Optimization (`reverse_linked_list.py`)
Solved LeetCode 206 (Reverse Linked List). Implemented an optimal $O(n)$ time and $O(1)$ space solution by iteratively manipulating node pointers. Used a temporary variable to safely traverse and reverse the direction of the list in-place.
