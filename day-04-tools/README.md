# Day 4: Function Calling Agent & Stack DSA 🛠️

## 🤖 1. AI Tool Agent (`tool_agent.py`)
Built an interactive agent capable of executing local Python functions. Implemented JSON schema definitions for tool mapping, handled AI tool call requests, executed local logic, and managed the 2-step API call loop to return dynamic results back to the LLM context.

## ⚡ 2. DSA Optimization (`valid_parentheses.py`)
Solved LeetCode 20 (Valid Parentheses) using a Python list to simulate a Stack (LIFO structure). Utilized the `pop()` method and dictionary hash maps to achieve an optimal $O(n)$ time and space complexity solution for validating nested structures.
