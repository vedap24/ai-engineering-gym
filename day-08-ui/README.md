# Day 8: Streamlit AI UI & BFS Queues 🖥️🔄

## 🤖 1. The AI Web App (`app.py`)
Transformed the terminal-based Agentic RAG system into a modern web application using Streamlit. Implemented conversation memory via `st.session_state` and rendered dynamic chat interactions using `st.chat_message` and `st.chat_input`. The UI successfully displays tool executions (Calculator and Vector Search) in real-time.

## ⚡ 2. DSA Optimization (`bfs_level_order.py`)
Solved LeetCode 102 (Binary Tree Level Order Traversal). Engineered a Breadth-First Search (BFS) algorithm using a Double-Ended Queue (`deque`). Processed the tree level-by-level using the FIFO (First-In, First-Out) principle to achieve an optimal $O(n)$ time complexity solution.
