## Day 2: Prompt Engineering & JSON Output Parsers

### 🎯 Objective
To force a Large Language Model to bypass conversational responses and strictly output machine-readable data (JSON) using advanced Prompt Engineering and Output Parsing.

### 🛠️ Concepts Explored
1. **System Prompts:** Instructing the AI on its specific persona (e.g., "Data Extraction Agent").
2. **JSON Mode:** Utilizing the `response_format={"type": "json_object"}` parameter to guarantee a structured dictionary output.
3. **Data Extraction:** Converting unstructured natural language (user text) into structured Key-Value pairs.

### 🧠 Associated DSA (Two Pointers)
Practiced the **Two Pointers** technique (LeetCode 125: Valid Palindrome) in the `dsa-patterns` directory. This technique is fundamental for efficient string manipulation and parsing algorithms, similar to how text is processed before tokenization.
