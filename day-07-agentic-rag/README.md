# Day 7: Ultimate Agent & Binary Trees 🌳⚙️

## 🤖 1. The Ultimate Agent (`ultimate_agent.py`)
Built a fully autonomous Agentic RAG system. The AI dynamically routes user queries between a math calculation tool and a vector database search tool by analyzing the JSON schema descriptions. It seamlessly falls back to normal conversation using `else` logic when no tools are needed.

## ⚡ 2. DSA Optimization (`tree_depth.py`)
Solved LeetCode 104 (Maximum Depth of Binary Tree). Implemented a recursive $O(n)$ solution to traverse `left` and `right` branches, calculating the maximum depth of a hierarchical tree structure.
